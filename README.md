[게임 설명]
- 고전 오락기 게임 보글보글에서 아이디어를 얻고 성벽을 지키는 디펜스 요소를 결합하여 학교 컨셉으로 제작함
- 적이 왼쪽 끝까지 도달하지 못하도록 방어하는 구조
- 적과 충돌하거나 적이 왼쪽 끝에 도달하면 생명 -1
- 스테이지 1~3으로 구성, 목표 점수는 10,000 / 15,000 / 20,000점
- 스테이지마다 적 캐릭터(맥주, 감기바이러스, 스마트폰)와 배경(정문, 정문-도서관 길, 도서관)을 다르게 구성
- 게임 시작 시 메뉴 화면에 튜토리얼을 포함하여 스토리와 조작 방법을 쉽게 이해할 수 있도록 함
